    CyberCursorOverlay 的行为说明。

    组件职责：

    • 监听鼠标移动、进入、离开、点击和窗口缩放事件
    • 在全屏 canvas 上绘制赛博风格鼠标拖尾粒子
    • 在点击时生成扩散涟漪
    • 在粗指针设备或用户偏好减少动画时自动降级

    使用方式：

    • 组件通常由 AppLayout 挂载，并通过 Teleport 渲染到 body 下
    • 当前版本不暴露 props、emits 或 slots
